\HTML5

This class offers convenience methods for parsing and serializing HTML5.

It is roughly designed to mirror the \DOMDocument class that is provided with most versions of PHP.

EXPERIMENTAL. This may change or be completely replaced.

load()

load(string $file) : \DOMDocument

Load and parse an HTML file.

This will apply the HTML5 parser, which is tolerant of many varieties of HTML, including XHTML 1, HTML 4, and well-formed HTML 3. Note that in these cases, not all of the old data will be preserved. For example, XHTML's XML declaration will be removed.

The rules governing parsing are set out in the HTML 5 spec.

Parameters

string $file

The path to the file to parse. If this is a resource, it is assumed to be an open stream whose pointer is set to the first byte of input.

Returns

\DOMDocument —

A DOM document. These object type is defined by the libxml library, and should have been included with your version of PHP.

The DOM to be serialized.

The filename to be written.
array $options

Configuration options when serializing the DOM. These include: - encode_entities: Text written to the output is escaped by default and not all entities are encoded. If this is set to TRUE all entities will be encoded. Defaults to FALSE.
